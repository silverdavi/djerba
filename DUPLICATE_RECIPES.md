These are the same dishes with different transliterations of Arabic/Hebrew names. They likely came from different recipe sources or used different transliteration systems.

## Source of Duplicates

These duplicates likely came from:
1. **Different transliteration systems**: Some sources use "aa" vs "a", "ou" vs "u", etc.
2. **Multiple recipe sources**: Recipes collected from different family members or cookbooks
3. **Regional spelling variations**: Different ways to write the same Arabic/Hebrew name in Latin script
