# מרק ירקות לקוסקוס (Merk Yerakot L'Couscous)

## רקע תרבותי
מרק ירקות לקוסקוס הוא מנה אהובה במסורת הקולינרית של היהודים הטוניסאיים, במיוחד בקרב הקהילה הדג'רבאית. המרק הצבעוני והטעים הזה, עשיר בירקות עונתיים ובתבלינים ארומטיים, מהווה תוספת מושלמת לקוסקוס, ומשקף מורשת תרבותית עמוקה המשלבת השפעות מהים התיכון וצפון אפריקה.

## הוראות שלב-אחר-שלב
1. **טיגון ארומטיים**: בסיר גדול, חממים את שמן הזית על חום בינוני. מוסיפים את הבצל הקצוץ ומטגנים עד שהוא שקוף, כ-5-7 דקות. מוסיפים את השום הקצוץ ומטגנים לעוד 1-2 דקות עד שהריח משתחרר.

2. **קליית תבלינים**: מוסיפים את הכמון הטחון, הכוסברה, הכורכום והג'ינג'ר לסיר. מערבבים היטב ומקלים את התבלינים במשך כ-1 דקה כדי לשחרר את הטעמים שלהם.

3. **הוספת ירקות**: מוסיפים את הגזר הקצוץ, הקישואים, תפוח האדמה, הדלעת והעגבניות הקצוצות לסיר. מערבבים היטב ומבשלים כ-5 דקות, מאפשרים לירקות להתרכך מעט.

4. **הוספת ציר וחומוס**: שופכים את ציר הירקות ומוסיפים את החומוס. מביאים לרתיחה, ואז מנמיכים את הלהבה. מכסים ומבשלים במשך 20-30 דקות, או עד שהירקות רכים.

5. **תיבול והתאמה**: טועמים את המרק ומוסיפים מלח, פלפל וחריסה לפי רמת החום הרצויה. אם משתמשים, מוסיפים את חלב הקוקוס לקרמיות נוספת.

6. **סיום עם לימון**: ממש לפני ההגשה, מוסיפים את מיץ הלימון הטרי כדי להאיר את הטעמים.

7. **הכנת קוסקוס**: בזמן שהמרק מתבשל, מכינים את הקוסקוס לפי הוראות האריזה או מבשלים אותו באופן מסורתי בקוסקוסייר. מפרידים את הקוסקוס היטב לאחר הבישול.

8. **הגשה**: בקערה, מניחים מנה נדיבה של קוסקוס ומוזגים את מרק הירקות מעל. מקשטים בכוסברה או פטרוזיליה טרייה.

## הצעות הגשה
מגישים את המרק חם עם לחם קריספי או סלט פשוט. מנה זו נהוגה כמנה קהילתית, מאחדת משפחה וחברים יחד.

## הקשר תרבותי והערות משפחתיות
מתכון זה בהשראת מסורות משפחת סילבר-כהן-טרבלסי, שבהן מרק ירקות לקוסקוס הוא מנה בסיסית במהלך שבת ואירועים חגיגיים. הוא משקף את שורשי המשפחה העמוקים במטבח הטוניסאי, חוגג גם את הטעמים של המורשת שלהם וגם את השמחה שבשיתוף ארוחות יחד.
